## Contact form email setup (EmailJS)

The contact form sends messages directly via EmailJS.

1. Copy `.env.example` to `.env.local`.
2. Fill the required variables from your EmailJS dashboard:
	- `NEXT_PUBLIC_EMAILJS_SERVICE_ID`
	- `NEXT_PUBLIC_EMAILJS_TEMPLATE_ID`
	- `NEXT_PUBLIC_EMAILJS_PUBLIC_KEY`
3. Configure template params in EmailJS: `from_name`, `reply_to`, `message`.
4. Restart the dev server.

Built-in client-side protection:
- Hidden honeypot field for basic bot filtering
